If you are an Acquia customer, you can manage your Acquia-hosted site via the Acquia Network web site. Use its management console to import your existing Drupal content or to install a Drupal distribution.

To visit the Acquia Network management console:

  1. Log in to your Acquia Network dashboard
  2. Select your site
  3. Click on the Dev Cloud or Managed Cloud link on the left site of the page
